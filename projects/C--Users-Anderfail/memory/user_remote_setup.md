User frequently works remotely through VS Code's integrated terminal to use Claude Code. Cannot easily view local browser windows in this setup. There is a Cloudflare tunnel configured for the appendix_k project website for remote access.

## Useful Claude Code features for remote workflow
- /remote-control: continue sessions across devices (ideal for VS Code remote setup)
- /btw: side-chain conversations for tangential questions without derailing main context
- /loop, /scheduled: recurring monitoring tasks (up to 3 days)
- Esc Esc (/rewind): undo bad directions instead of fixing in same context
